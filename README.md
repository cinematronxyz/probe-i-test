INSTALACIJA AR SOFTVERA (UBUNTU 24.04 FIX)
------------------------------------------

Ova verzija instalacije je prilagođena za Ubuntu 24.04. Umesto direktne 
instalacije starog .deb paketa (koji ne radi na novom sistemu), ova 
skripta instalira zvanični DDD i "ubacuje" LongU/LongH 
prečice i registre.

UPUTSTVO ZA INSTALACIJU:

1. www.acs.uns.ac.rs/sr/node/237/594 --> instalirati .tar.gz za verziju 22.04. i extract-ovati ga
2. Desni klik u folderu gde se nalaze ovi fajlovi -> "Open in Terminal".
3. Dozvolite pokretanje skripte komandom:
   
   chmod +x skripta.sh

4. Pokrenite instalaciju:

   ./skripta.sh

5. Unesite vašu lozinku (neće se videti zvezdice dok kucate) i sačekajte.
6. Kada se završi, ukucajte ddd u terminal da vidite da li radi, ako ne odjavite se sa sistema i ponovo prijavite (Log out/Log in).
